目录与配置说明

data/
- 保存抓取与生成的CSV数据
- 包含：
-  持仓分析YYYYMMDD.csv
-  增强涨停YYYYMMDD.csv
-  增强跌停YYYYMMDD.csv
-  板块资金与涨幅.csv
-  cache_market/ 市场情绪缓存

reports/
- 保存每日分析报告MD文件
- 如：每日分析报告YYYYMMDD.md

根目录配置
- concept_config.json：概念黑名单与评分参数
- 持仓股票.csv：每日持仓输入，格式：日期,股票名称[,持仓成本]

生成方式
- uv run python -m fupan.board_params_fetch --date YYYYMMDD
- uv run python -m fupan.fupan --date YYYYMMDD
